# Am I Even Or Odd 🤔

A tiny Python package that answers one simple question.

## Installation

```bash
pip install am-i-even-or-odd
```

## Usage

```python
from am_i_even_or_odd import is_it_even_tho, is_it_odd_tho

print(is_it_even_tho(4))
print(is_it_odd_tho(5))
```

Output

```
True
True
```
